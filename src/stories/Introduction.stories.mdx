## 설치 전 준비사항
- NodeJS<br/>
    LTS 설치 파일을 다운로드 받아서 설치<br/>
    https://nodejs.org/ko/download

- Git Bash<br/>
    https://git-scm.com/downloads


## 설치
> 순서대로 아래 명령어를 입력하시면 프로젝트 개발 환경을 구성할 수 있습니다.

<br/>

#### 1. Yarn 설치
```shell
npm install -g yarn
```

#### 2. 프로젝트 설치 받을 경로로 이동
```shell
# 예시
cd $HOME
```

#### 3. 프로젝트 다운로드
```shell
git clone http://192.168.10.80:30000/opflow/opflow2-coreui.git
```

#### 4. 패키지 설치
```shell
yarn install
```

#### 5. 프로젝트 실행
```shell
yarn start
```
